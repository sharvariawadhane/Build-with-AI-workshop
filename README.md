# Local-First File Organizer

## Overview
A browser-based file organizer that stores and manages files locally using IndexedDB. No cloud upload.

## Tech Stack
- Next.js
- Tailwind CSS
- IndexedDB (Dexie)

## Features
- Upload and store files locally  
- Tag and categorize files  
- Search and filter
- write notes for each file
- AI auto-tagging
- Drag & drop
